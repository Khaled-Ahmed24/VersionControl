# Version Control Learning Repository

This repository is a collaborative learning project for the **ITI 9-Month Program**, focused on practicing **Git and version control** concepts.

## 📋 Overview

This project serves as a hands-on practice environment where team members learn and demonstrate:
- Git fundamentals (commits, branches, merges)
- Collaborative workflows
- Conflict resolution
- Branch management

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd VersionControl
   ```

2. **View all branches:**
   ```bash
   git branch -a
   ```

3. **Switch to a specific branch:**
   ```bash
   git checkout <branch-name>
   ```

## 📝 Contributing

Each team member maintains their own branch. To contribute:
1. Create or switch to your assigned branch
2. Make changes to your files
3. Commit with meaningful messages
4. Push changes to the remote repository
5. Create a pull request for review if needed

## 📚 Learning Objectives

- Understanding Git workflows

- Learning merge strategies
- Resolving merge conflicts
